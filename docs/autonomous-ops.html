Configuration Examples

How autonomous tasks will be defined

GitHub Webhook Handler

# fleet-webhooks.yaml webhooks: on_push: branch: main tasks: - update-docs - run-linter on_pr_opened: tasks: - generate-tests - security-scan - code-review on_issue_labeled: label: bug tasks: - investigate-bug

Scheduled Tasks

# fleet-schedule.yaml schedules: nightly_maintenance: cron: "0 2 * * *" tasks: - update-dependencies - fix-lint-errors - optimize-imports weekly_audit: cron: "0 9 * * MON" tasks: - security-audit - coverage-report - tech-debt-analysis

Task Template

# templates/update-docs.yaml name: update-docs description: Update documentation role: tech-writer steps: - analyze-changes - update-readme - generate-api-docs - create-pr on_complete: notify: slack:#docs

Multi-Repo Config

# fleet-repos.yaml repositories: frontend: url: github.com/org/frontend default_branch: main workers: 2 backend: url: github.com/org/backend default_branch: develop workers: 3 shared-libs: url: github.com/org/libs workers: 1
