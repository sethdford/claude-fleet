Capabilities

What Claude Fleet does

Production infrastructure for coordinating multiple Claude Code instances.

Worker Orchestration

Spawn and manage multiple Claude Code instances. Each worker runs in an isolated git worktree, preventing conflicts.

Implemented

Blackboard Messaging

Shared context propagation via blackboard messaging. Workers post discoveries, others subscribe to updates.

Implemented

Real-Time Dashboard

Monitor worker health, task progress, and fleet metrics through a WebSocket-powered web dashboard.

Implemented

JWT Authentication

Role-based access control with team-lead and worker roles. Secure API endpoints with token-based auth.

Implemented

SQLite Persistence

Workers, checkpoints, messages, and work items persisted to SQLite. Automatic crash recovery.

Implemented

MCP Integration

Model Context Protocol server for seamless integration. Control your fleet from any MCP-compatible client.

Implemented
